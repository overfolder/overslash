# Review

This is a review of the project by human Overs/ash creators

## Problems

+ Remove Dashboard, make Agents the default view
+ The Identities view should be called: Agents
+ The Agents view should show the User Agent tree, it doesnt make sense to allow adding other User identities. The root of the tree should be the user, unmutable
+ Create identity: the Kind dropdown makes no sense, this button should only allow creating Agents. SubAgents and Agents not different things, only different is what parents do they have
+ dark mode accents on hover, pills such as "inherit" and many others are low visible
+ the delete confirmation is an ugly browser popup, make it a well designed modal
+ i dont see a template editor. i think it is on the backlog
+ API Explorer is also on backlog? i cant test the overslash meta-service with it
+ Org Settings page says: Cannot load org settings. Admin access required to view org settings.
  even when using Dev User
